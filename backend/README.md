### **Installation**

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd jubilant-octo-fortnight/backend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Setup SQLite database**

   ```bash

   ```

   Navigate to `http://localhost:3000`
